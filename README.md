# Gerenciador de Lista de Tarefas em Swift

## Descrição

Aplicação de linha de comando desenvolvida em Swift que permite gerenciar uma lista de tarefas diretamente pelo terminal.

O programa implementa as operações básicas de um CRUD:

- Criar tarefas
- Listar tarefas
- Editar tarefas
- Remover tarefas

Este projeto foi desenvolvido com foco em praticar lógica de programação, manipulação de arrays e organização de código com separação de responsabilidades.

---

## Funcionalidades

O usuário pode:

1. Listar todas as tarefas cadastradas
2. Adicionar uma nova tarefa
3. Remover uma tarefa existente
4. Editar uma tarefa
5. Encerrar o programa

O sistema valida entradas do usuário para evitar erros de execução.

---

## Tecnologias Utilizadas

- Swift
- Foundation
- Execução via terminal (CLI)

---

## Conceitos Aplicados

- Funções

- Estruturas condicionais
